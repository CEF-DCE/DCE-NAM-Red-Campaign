# DCE-NAM-Red-Campaign
DCS World DCE Vietnam war on the Red Side Campaign
NAM 1965-74 - This is a semi-historical campaign meant to represent a certain vision of the Vietnam War. This is the North Vietnamese version of the campaign, which promises to be difficult against the US armada. Aircraft and weaponry will evolve throughout this campaign, as will the authorized targets for the US.
